# Use Bun Instead of npm

Use [Bun](https://bun.sh/) as the package manager and task runner for this project instead of npm.

**Install dependencies:**
```bash
bun install
```

**Run scripts:**
```bash
bun run <script-name>
```

**Add packages:**
```bash
bun add <package-name>
```

Bun provides faster performance and better compatibility with Node.js projects.

## Additional Rules
- Do not run build or run dev, assuming it is user work
